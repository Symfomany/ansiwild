Ansiwild
=========

Ansible Role for install Mysql, Mongo, Apache2, Composer, NodeJS, NPM, Git, PHP7, PHP CLI, Curl with:

Modules:

- php7.0-common
- php7.0-cli
- php7.0-dev
- php7.0-gd
- php7.0-curl
- php7.0-json
- php7.0-intl
- php7.0-opcache
- php7.0-xml
- php7.0-exif
- php7.0-mbstring
- php7.0-mysql
- php-pear
- mysql-client
- mysql-server

Configure with:

- Virtualhost
- Mode Rewrite
- Timezone Europe/Paris
- utf-8 mysql/php
- short_open_tag = off
- memorit_limit 512M 


Requirements
------------

Ubuntu 16.04 or 16.10 Server/Destop
